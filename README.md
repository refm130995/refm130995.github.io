# refm130995.github.io

Personal website of **Ramon Eduardo Figuera** — Senior Full-Stack Developer.

**Live:** https://refm130995.github.io/

Built with React + TypeScript + Vite. A "technical dossier" design — warm espresso palette, Fraunces display with JetBrains Mono metadata, and restrained motion (a staggered hero entrance, scroll reveals, refined hover states) that respects `prefers-reduced-motion`. No animation libraries.

## Run locally

```bash
npm install
npm run dev
npm run build
```

Deployed to GitHub Pages via GitHub Actions on every push to `main`.
